# Python-Assignment-1
Attendance Tracker Program
This system helps you record students' names along with their check-in time, prevents duplicate entries, and automatically generates a detailed summary. It allows users to record attendance with arrival times, calculates the number of absentees, and exports the complete attendance record to a CSV file for easy sharing and storage.


Features:
•	Allows adding multiple students at once
•	Prevents duplicate attendance entries
•	Generates a clean, formatted attendance summary
•	Calculates total present and absent students
•	Saves the attendance record to a CSV file for convenient sharing and long-term storage
